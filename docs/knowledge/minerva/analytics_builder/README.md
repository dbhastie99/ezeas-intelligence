# Analytics Builder Knowledge Pack

This directory contains the static Minerva knowledge-pack index for the Analytics Builder Guide corpus.

The pack is metadata-only. It registers expected source groups from `ezeas-analytics` static OMG v0.2 and links them to M1/M2 Minerva metadata. It does not copy source content, does not copy generated HTML, and does not perform runtime ingestion.

Production answer use remains blocked pending governed source ingestion.
