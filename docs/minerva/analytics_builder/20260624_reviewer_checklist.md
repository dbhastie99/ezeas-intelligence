Use this checklist when reviewing the M8 static demo readiness pack.

## Source Lineage

- Each question identifies required source artifact groups.
- Generated HTML remains reference-only.
- Source import and answer evaluation are still pending.

## Safety Wording

- Final-paid payroll truth remains UNPROVEN / Blocked.
- PayrollLedger is not bank-paid proof.
- CalcInterpreterLine is calculation/detail evidence, not payment execution.
- ObjectTime is source-context evidence, not payment finality.
- PayRun finalisation or SUCCEEDED status alone does not prove settlement, bank acceptance, remittance, or final-paid truth.
- Visual rendering is not certification proof.

## Prohibited Claims

- No production answer use claim.
- No production-passed baseline claim.
- No Certified asset claim.
- No final-paid proof claim.
- No bank-paid, settlement, remittance, or payment-finality shortcut claim.

## Readiness

- Demo status is static non-production readiness.
- Baselines are planned/pending only.
- Runtime answer generation is not enabled.
- Governed source ingestion and answer evaluation remain blockers.

## Blocked Gaps

- Blocked gaps are safety controls, not failures.
- Missing upstream proof is stated rather than invented.
